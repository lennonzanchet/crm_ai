# 🚀 CRM AI

Um CRM moderno desenvolvido com Node.js, TypeScript, Fastify, Prisma e PostgreSQL, projetado para ser escalável, performático e preparado para recursos de Inteligência Artificial.

Este projeto está sendo desenvolvido seguindo boas práticas de arquitetura, versionamento e organização de código, evoluindo etapa por etapa.

# ⚙️ Como executar

## 1. Clonar o projeto

```bash
git clone https://github.com/SEU_USUARIO/crm-ai.git
```

## 2. Entrar na pasta

```bash
cd crm-ai
```

## 3. Instalar dependências

```bash
npm install
```

## 4. Configurar as variáveis de ambiente

```bash
cp .env.example .env
```

## 5. Subir o banco de dados

```bash
docker compose up -d
```

## 6. Iniciar o servidor

```bash
npm run dev
```
